# donboigi 컨셉북

> **donboigi** = 돈(money) + 돋보기(magnifying glass)
> 공공데이터와 금융 데이터를 쉽게 들여다보는 데이터 연구소

---

## 1. 프로젝트 개요

donboigi는 공공데이터포털(data.go.kr)과 금융 관련 API 데이터를 수집·가공·시각화하여 일반 사용자도 쉽게 탐색할 수 있는 데이터 연구소 웹앱이다.

- 복잡한 공공·금융 데이터를 직관적인 대시보드로 제공
- Go 단일 바이너리로 배포하여 운영 부담 최소화
- 광고 기반 수익 모델로 무료 서비스 운영

---

## 2. 기술 스택

| 영역 | 기술 | 비고 |
|------|------|------|
| **프론트엔드** | React + TypeScript + Vite | SPA, 빠른 빌드 |
| **스타일링** | Tailwind CSS | 유틸리티 퍼스트 |
| **UI 컴포넌트** | shadcn/ui | 커스터마이즈 가능한 Radix 기반 컴포넌트 |
| **백엔드** | Go (net/http 또는 Echo/Chi) | 고성능, 단일 바이너리 |
| **데이터베이스** | SQLite | 파일 기반, 임베디드 |
| **배포 방식** | `go:embed` | 프론트 빌드 산출물을 Go 바이너리에 임베드 |

### 단일 바이너리 전략

```
[Vite 빌드] → dist/ → go:embed → 단일 실행 파일
```

프론트엔드 빌드 결과물을 Go 바이너리에 포함시켜 별도 웹 서버 없이 하나의 실행 파일로 서비스한다.

## 3. 타겟 데이터 소스

> 상세 조사 문서: **[DATA_SOURCES.md](./DATA_SOURCES.md)**

### 공공데이터포털 (data.go.kr)

- 국토교통부 실거래가 정보 (아파트/오피스텔/상업용 매매·전월세)
- 금융위원회 주식발행정보, 증권상품시세정보
- 국세청 사업자등록 상태 조회
- 통계청 온라인 수집 가격 정보
- 한국수출입은행 환율 정보
- 금융감독원 공시정보 (기업개황, 고유번호)

### 독립 금융 API 포털

- 한국은행 ECOS API (기준금리, 환율, GDP, 통화량)
- 금융감독원 OpenDART (재무제표, 공시, 지분변동)
- 금융감독원 금융상품한눈에 (예적금·대출 금리 비교)
- 한국석유공사 오피넷 (유가 정보)
- 통계청 KOSIS (CPI, 고용, 인구 통계)

> 데이터 소스는 서비스 확장에 따라 점진적으로 추가한다.

## 4. 주요 기능 구상

### Phase 1 — MVP

- **데이터 탐색기**: 공공데이터를 카테고리별로 탐색, 테이블/차트 뷰 제공
- **실거래가 조회**: 아파트·오피스텔 실거래가 검색 및 시계열 차트
- **환율/금리 대시보드**: 주요 환율·기준금리 추이 시각화
- **기업공시 뷰어**: DART 공시 데이터 요약 조회

### Phase 2 — 확장

- **비교 분석 도구**: 지역별·기간별 데이터 비교
- **알림 설정**: 특정 조건 충족 시 알림 (가격 변동, 공시 등)
- **데이터 다운로드**: CSV/Excel 내보내기
- **사용자 대시보드 커스터마이징**: 관심 데이터 위젯 구성

### Phase 3 — 고도화

- **AI 요약**: LLM 기반 데이터 트렌드 요약
- **예측 모델**: 간단한 시계열 예측 시각화
- **커뮤니티**: 사용자 간 분석 결과 공유

## 5. 수익 구조

### 광고 모델

| 광고 유형 | 위치 | 설명 |
|-----------|------|------|
| **배너 광고** | 사이드바, 하단 | Google AdSense 등 디스플레이 광고 |
| **네이티브 광고** | 데이터 카드 사이 | 콘텐츠와 자연스럽게 어우러지는 광고 |
| **스폰서 콘텐츠** | 분석 리포트 | 금융사·핀테크 기업 협찬 리포트 |

### 수익화 원칙

- 핵심 데이터 조회 기능은 무료 유지
- 광고가 사용자 경험을 해치지 않는 선에서 배치
- 추후 프리미엄 구독(광고 제거, 고급 분석) 모델 검토 가능

## 6. 향후 확장 계획

| 단계 | 기술/서비스 | 목적 |
|------|-------------|------|
| **DB 복제** | Litestream | SQLite를 S3 등에 실시간 스트리밍 백업 |
| **인증/BaaS** | Supabase | 사용자 인증, 실시간 구독, 스토리지 |
| **캐싱** | Redis 또는 SQLite 캐시 테이블 | API 호출 최소화, 응답 속도 개선 |
| **모니터링** | Prometheus + Grafana | 서비스 메트릭 수집 및 시각화 |
| **CDN** | Cloudflare | 정적 자산 캐싱, DDoS 방어 |
| **컨테이너화** | Docker | 배포 자동화, CI/CD 연동 |
